# Data Visualization using Python

## Project Overview

This project focuses on transforming raw Superstore sales data into meaningful visual insights using Python visualization libraries. The analysis helps understand sales trends, profitability, regional performance, discount impact, and customer behavior through charts and graphical representations.

The project demonstrates how data visualization supports data-driven decision-making in business environments.

---

## Objectives

* Analyze sales and profit performance
* Identify trends and seasonal patterns
* Understand the impact of discounts on profitability
* Compare regional and category-wise sales
* Create meaningful and professional visualizations
* Generate actionable business insights

---

## Dataset

Dataset Used: Superstore Sales Dataset

The dataset contains:

* Order details
* Customer segments
* Product categories
* Sales and profit values
* Discount information
* Regional performance data

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Features Implemented

### Data Cleaning and Preprocessing

* Handled date formatting
* Created new analytical columns
* Feature engineering for profit margin and discount analysis

### Visualizations Created

* Sales by Region
* Category-wise Sales and Profit Margin
* Monthly Sales Trend
* Yearly Sales Growth
* Discount vs Profit Analysis
* Profit Margin Distribution
* Heatmap Analysis
* Correlation Matrix
* Pairplot Analysis
* Sub-category Performance Analysis

---

## Key Insights

* West region generated the highest sales
* Technology category contributed maximum revenue
* High discounts reduced profitability
* Sales increased during festive and year-end seasons
* Some product sub-categories produced better profit margins

---

## Business Recommendations

* Avoid excessive discount strategies
* Focus marketing efforts on high-performing regions
* Improve sales performance in underperforming regions
* Promote high-margin products
* Plan inventory before seasonal sales peaks
